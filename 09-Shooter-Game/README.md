# Shooter Game

## Description
This project is an assignment that utilizes the Unity Hub Application to explore the Physics functions available in the app and create a Shooter Game.

## Technologies Used
- Python
- Deep Learning
- CNN
- CNN Structured Denoising Method
- U-Net Segmentation
